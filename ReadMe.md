# Installation and running

**Installation:**
In project ./client
```bash
npm install
```

**Running:**  
Navigate to the `./client` directory and install the dependencies:

```bash
npm run dev
```

**App will start on port:**
| App | URL |
|------------------|-------------------------------------|
| Front-end | [http://localhost:3000](http://localhost:3000) |
