# Installation and running



**Installation:**  

```bash
npm install
```

**Running:**  
To run the UI, use the following command in this folder:
```bash
npm run dev
```

**App will start on port:**
| App              | URL                                 |
|------------------|-------------------------------------|
| Front-end | [http://localhost:3000](http://localhost:3000) |
